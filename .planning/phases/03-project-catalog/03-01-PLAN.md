**Step 2 — Update app/page.tsx**

The page needs two sections:
1. The existing splash (preserve exactly — do not alter the heading or blurb)
2. A new catalog section below the splash

Change `min-h-screen flex flex-col items-center justify-center` on `<main>` to `min-h-screen flex flex-col items-center` — remove `justify-center` so the page can scroll naturally with content below the splash. The splash div gets `flex-1` and `justify-center` to stay centered in its allocated space.

Full replacement for `app/page.tsx`:

```tsx
import { projects } from "@/lib/projects";

export default function Home() {
  return (
    <main className="min-h-screen flex flex-col items-center px-6">
      {/* Splash — vertically centered in viewport */}
      <div className="flex-1 flex flex-col items-center justify-center gap-6 text-center max-w-lg w-full">
        <h1 className="heading-gradient lowercase tracking-tight leading-none">
          the shadow realm
        </h1>
        <p className="text-[#ededed]/60 text-sm tracking-wide">
          my digital junk drawer.
        </p>
      </div>

      {/* Project catalog */}
      <section className="w-full max-w-lg pb-24">
        <h2 className="text-[#ededed]/40 text-xs uppercase tracking-widest mb-8">
          projects
        </h2>
        <ul className="flex flex-col gap-6">
          {projects.map((project) => (
            <li key={project.slug}>
              <a
                href={project.href}
                className="block group"
              >
                <span className="text-[#ededed] text-sm font-medium group-hover:text-[#c8cdd4] transition-colors duration-0">
                  {project.name}
                </span>
                <p className="text-[#ededed]/40 text-sm mt-1">
                  {project.description}
                </p>
              </a>
            </li>
          ))}
        </ul>
      </section>
    </main>
  );
}
```

**Style notes (per D-15, D-16, D-17):**
- Background stays `#0a0a0a` via body in globals.css — no inline bg needed
- Text: `#ededed` for entry name, `#ededed]/40` for description and section label
- Section label "projects" uses `text-xs uppercase tracking-widest` — subtle, lowercase-adjacent vibe
- `duration-0` on the transition-colors effectively disables the animation (renders instantaneous) — satisfies D-16 (no animations/transitions) while keeping hover state accessible
- No border, no card box, no tags, no tech labels — per D-04, D-05
- `pb-24` gives comfortable breathing room below the last entry

**Import:** Use `@/lib/projects` — the `@` alias maps to project root per Next.js 15 default tsconfig.

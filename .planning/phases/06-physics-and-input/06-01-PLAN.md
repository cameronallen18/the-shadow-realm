**1. Type exports:**

```typescript
export interface Obstacle {
  x: number;         // left edge, CSS pixels
  gapTop: number;    // y where gap starts (CSS px from top)
  gapBottom: number;  // y where gap ends (CSS px from top)
  scored: boolean;   // true once Samus has passed this obstacle (prevents double-count)
}

export interface GamePhysicsState {
  samusY: number;          // CSS px from top of canvas (0 = ceiling)
  samusVY: number;         // CSS px/s, positive = downward
  obstacles: Obstacle[];
  obstaclesCleared: number;
  speedMultiplier: number;
  gameOver: boolean;
  pendingJump: boolean;    // set true on first input; consumed by first loop frame
}
```

**2. Factory function:**

```typescript
export function createInitialGameState(canvasWidth: number, canvasHeight: number): GamePhysicsState
```

Implementation:
- Set `samusY` to `canvasHeight * GAME.floorRatio` (starts on floor)
- Set `samusVY` to `0`
- Create 2 obstacles using `randomGap(canvasHeight)`:
  - Obstacle 0: `x = canvasWidth + 100`
  - Obstacle 1: `x = canvasWidth + 100 + canvasWidth * OBSTACLE_SPACING_RATIO`
  - Both with `scored: false`
- Set `obstaclesCleared: 0`, `speedMultiplier: 1`, `gameOver: false`, `pendingJump: false`

**3. randomGap helper (not exported):**

```typescript
function randomGap(canvasHeight: number): { gapTop: number; gapBottom: number }
```

Implementation:
- `playAreaTop = canvasHeight * 0.1`
- `playAreaBottom = canvasHeight * GAME.floorRatio`
- `playHeight = playAreaBottom - playAreaTop`
- `gapSize` = random between `playHeight * 0.28` (min) and `playHeight * 0.45` (max)
- `gapCenter` = random position within valid range so gap fits between playAreaTop and playAreaBottom
- Return `{ gapTop: gapCenter - gapSize / 2, gapBottom: gapCenter + gapSize / 2 }`

**4. updateGame function:**

```typescript
export function updateGame(state: GamePhysicsState, dt: number, canvasWidth: number, canvasHeight: number): void
```

This function mutates `state` in place. Steps in order:

a. **Pending jump** — if `state.pendingJump` is true, set `state.samusVY = -PHYSICS.jumpVelocity` and `state.pendingJump = false`

b. **Gravity** — `state.samusVY += PHYSICS.gravity * dt`, then clamp: `state.samusVY = Math.min(state.samusVY, PHYSICS.terminalVelocity)`

c. **Position** — `state.samusY += state.samusVY * dt`

d. **Floor clamp** — if `state.samusY >= canvasHeight * GAME.floorRatio`, set `state.samusY = canvasHeight * GAME.floorRatio` and `state.samusVY = 0`

e. **Ceiling clamp** — if `state.samusY <= 0`, set `state.samusY = 0` (no gameOver yet — Phase 7)

f. **Obstacle scrolling** — compute `speed = PHYSICS.baseScrollSpeed * state.speedMultiplier`. For each obstacle:
   - `obs.x -= speed * dt`
   - If `obs.x + GAME.obstacleWidth < 0` (fully off left edge):
     - Reset `obs.x = canvasWidth + 50` (re-enter from right)
     - Assign new gap via `randomGap(canvasHeight)`
     - Set `obs.scored = false`
     - Increment `state.obstaclesCleared`
     - If `state.obstaclesCleared > 0 && state.obstaclesCleared % 10 === 0`: increment `state.speedMultiplier += PHYSICS.speedIncrement`, cap at `PHYSICS.maxSpeedMultiplier`

No collision detection (deferred to Phase 7). No gameOver trigger (Phase 7).

```typescript
export function triggerJump(state: GamePhysicsState): void {

}
```

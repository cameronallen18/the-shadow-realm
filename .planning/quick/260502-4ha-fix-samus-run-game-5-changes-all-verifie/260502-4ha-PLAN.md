### Task 2: Add gap difficulty curve to game loop
**Files:** `components/samus-run/gameLoop.ts`

**Action:**
1. Add `obstaclesSpawned: number` to the `GamePhysicsState` interface
2. Replace `randomGap(canvasHeight)` with `gapForObstacle(n: number, canvasHeight: number)`:
   - `n <= 5`: `gapSize = maxGap` (first 5 always max size, easiest)
   - `6 <= n <= 15`: `gapSize = minGap + Math.random() * (maxGap - minGap) / 2` (half range)
   - `n > 15`: `gapSize = minGap + Math.random() * (maxGap - minGap)` (full range, current behavior)
   - Keep existing center randomization logic unchanged
3. In `createInitialGameState`: set `obstaclesSpawned: 2`, use `gapForObstacle(1, h)` and `gapForObstacle(2, h)` for the two initial obstacles
4. In `updateGame` recycle logic: increment `state.obstaclesSpawned` BEFORE calling `gapForObstacle`, then call `gapForObstacle(state.obstaclesSpawned, canvasHeight)`

**Verify:** TypeScript compiles. `obstaclesSpawned` is tracked. Recycle logic calls `gapForObstacle`.

**Done:** No TypeScript errors in gameLoop.ts.
